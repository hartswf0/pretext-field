ARGO · Operative Cartography

A shared control surface for spatial node graphs. Navigate, inspect, and operate on world nodes through four interconnected views.

Views

VORONOI3D perspective of the territory map. Drag nodes, hover to inspect, click to select.
FLATTop-down Voronoi map. Click any territory to teleport. Same tessellation as VORONOI.
WALKFirst-person raycaster. Explore walls of text. Minimap shows position on the Voronoi.
SPHERE360 photosphere interior for nodes with environments.

Controls

W/A/S/DMove (walk mode) or navigate (flat mode)
A/DTurn left/right (walk mode)
1-9Teleport to node by index
GFire operator on selected node
PToggle spatial prompting (walk mode — auto-generate on territory crossing)
HToggle grid overlay (voronoi mode)
Ctrl+ZUndo (restore previous state snapshot)
Ctrl+Shift+ZRedo
Double-clickReset to full overview (deselect + zoom out)
ESCClose panels
MouseOrbit (voronoi), click territory (flat), drag nodes (voronoi)
ScrollZoom in/out

Panels (A/B/C/D)

A · INSPECTNode inspector with type, affinity, photosphere access
B · BUILDAdd nodes, set API key for image/LLM generation
C · GRAPHExport data, graph statistics
D · EXPORTWorldText export and timeline

API / Rate Limits

OpenAI rate limits reset per-minute (~60s). The system auto-retries on 429 errors. Image generation: gpt-image-1 ($0.04/img). Text analysis: gpt-4.1-mini. Set a budget cap in panel B to prevent overages.

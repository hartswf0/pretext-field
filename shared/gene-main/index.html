Vision & Philosophy

Conventional software imposes structure. It provides tools that, while powerful, subtly guide creativity down pre-defined paths. This project is an experiment in escaping that paradigm.

We treat content not as a static artifact, but as a living organism to be cultivated. By composing information as a 'genome,' the creative process shifts from mechanical assembly to guided evolution, fostering emergent and unexpected results.

Key Features

  • Visual Genome Editor: Drag-and-drop codons to build your content sequence.
  • Live Phenotype Preview: See your changes rendered in real-time.
  • Codon-Based Content: Use discrete codons for text, images, styles (CSS), and logic (JS).
  • Capture Phenotype: Export your final creation as a self-contained HTML file.
  • Recombination Lab: Experiment with advanced genetic operators in the editor.
  • Mutation Engine: Apply random changes to discover emergent designs.
